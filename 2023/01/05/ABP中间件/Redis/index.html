Redis

  • 使用Redis步骤

    • 虚拟机1 安装redis 客户端

    • 修改redis配置文件

    • 运行redis 端口 6379

    • 虚拟机1 启动redis客户端，连接对应地址

    • Vs下载stackerchange.Redis nuget包

    • 封装Rdis使用的扩展方法类

    • starup类文件添加Redis中间件

    • 控制器文件注入 ConnectionMultiplexer 服务

    • redis 换区一条数据，没有则查询，并存入Redis

    • Redis hashSet存储（字典）,实现存储指定值，并自增，自减

    • redis 实现分页查询

    • redis 存储集合列表

    • redis 实现事务存储

    • 加入静态锁，防止超卖

    • 加入分布式锁，防止集群超卖

redis 主从集群

  • 缺点：只有一台主机，不能实现高可用

redis 哨兵集群

  • 缺点：无法高并发读写，无法处理海量数据

redis-cluster

  • 缺点：消耗资源
  • 优点： 高并发读写，，高可用，存储海量数据
